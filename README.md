# RandomNumberCount
A data pipeline is about "Random Number Counting"

# Buidling package
```
mvn clean package
```

# Building containers

1. Building docker file
```
docker build -t kafka-spark-flink-example .
```

2. Building from docker-compose file.
```
docker-compose up -d
```
